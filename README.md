## 简介

Terminal插件提供了远程终端接入能力，在用户既有权限内更加方便的进行终端操作。 

主要功能：

1. 多窗口/多标签操作
2. 支持空闲会话回收
3. 支持录屏&回放
4. 支持实时终端输入的高危命令检测(依赖高危命令插件)
5. 文件上传下载的权限控制
